## Reeco

Reeco is a Dining Concierge chatbot, that sends you restaurant suggestions given a set of preferences that you provide the chatbot withthrough conversation.


### AWS Services

* SAM
* API Gateway
* Lambda Functions
* Amazon Lex
* SQS
* Elastic Search
* DynamoDB
* AWS Cloudfront
* IAM
* SNS
